CloudMessage

This is when I decided to try my hand at cloud data. In Scratch, cloud variables can only be numbers. This meant I had to convert letters into number and vice versa. I had made an earlier version, but it had a maximum letter capacity of 5, and it was much harder to add characters. This version can have as meany characters as you want, and uses lists and custom block to make the process more efficient. Press space to change the message. This is not a chat. It is more of a billboard which anybody can write on. WARNING: THIS IS NOT FILTERED. ANYTHING WRITTEN ON HERE IS THE RESPONSIBILY OF THE PERSON WHO WROTE IT. TO REPORT RUDE MESSAGES, EMAIL ME. Note: if you are not a Scratch user, you cannot edit the message. If you want to edit the message, you must sign up to scratch, and you must become a Scratcher (not a new Scratcher). If you are already one, you must be signed in in the browser. If it doesn't work here, try going to the actual project.

SelfieMaker

To save the selfie: Click this link, Click 'See inside', Right click on the stage (the big window in the top left corner), Click 'save picture of stage, Select the file destinaton (and name if you want) and click 'Save'. FUNCTIONS: Press 'm' to select the moustache, Press 's' to select the sunnies, Press 'b' to select the beard, Press the arrow keys to move the selected item, Press '0' to make the selected item dissapear, Press 1-3 to change the look of the selected item, Press 'n' to display your usename in the selfie, Press 'x' to hide your usernname, Press 'i' and 'k' to change the filter (filters 1-7 are the same colour but different exposure - 1=brightest 7=darkest. Personally I use 7, but it varies in different light), Press 'p' to hide the variables 'Selected' and 'Filter' so it is not in your selfie, Press 'd' once you have taken your selfie and want the 'Selected' and 'Filter' variables back. YOU NEED A WEBCAM FOR THIS TO WORK.
